1. General vocabulary
(a) What is a parameter and an argument?
    Parameters are defined and required by the function.
    Arguments are variables passed to the function that must match the defined paramters.
(b) In relation to Arrays, what is an element and what is an index?
    An element is a value/variable in the array. Index or key can be used to recall elements in the array.

(b) What is the function to remove the top item from the stack?
    pop

(d) How are stacks and queues different?
